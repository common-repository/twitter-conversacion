== Description ==

Widget de búsqueda de Twitter como widget de wordpress. Una vez instalado como plugin (ya sabes lo subes por ftp a tu carpeta plugins y en el panel de administración lo activas) sólo tienes que arrastrarlo al sitio deseado de tu barra lateral y editarlo.
Tienes que introducir tu nombre de usuario de Twitter y puedes editar desde el color del texto o de fondo de la cabecera y cuerpo, hasta las dimensiones, si quieres que se vean o no los avatares, o los titulares del widget, etc.
